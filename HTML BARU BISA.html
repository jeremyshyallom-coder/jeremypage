Tentang Saya

Nama Saya Jeremy Tambunan dan saya adalah mahasiswa DKV Universitas Binus (Undergraduate). Saya punya pengalaman freelance dalam proyek desain grafis dan branding selama setahun. Saya bekerja di cafe @niftysip sebagai desainer grafis sekaligus perumus visual brand identity. Saya cukup banyak pengalaman dengan klien dalam produksi graphic standar manual dan artwork. Saya senang dengan bagaimana sebuah grafis bisa diproduksi, dan saya senang mengeksplorasi hal-hal baru.
